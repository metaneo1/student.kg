# student.kg
Service for tutoring in Kyrgyzstan. The purpose of this project is to create a website for tutors that connects tutors and students, parents who are in need a tutor for their children or for themselves. This project is new in Kyrgyzstan and itâ€™s the only one that was created. The service has been implemented under the guidance of Alatoo-Enactus team. 
Student.kg website will be connecting tutors and who ever in need of them in a simple way. If this project is successfully launched it will take a positive difference by helping the society to find or communicate with tutors and make education more available.
